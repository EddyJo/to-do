## 상태 정의

| 상태 | 설명 |
|------|------|
| draft | 초안 작성 중, 미승인 |
| in-progress | 실행 중 |
| review | PR 리뷰 대기 |
| done | 완료 및 병합됨 |
| cancelled | 취소됨 (이유 명시) |

## 계획 생성 절차

1. `template.md` 복사 → `PLAN-XXX.md` (번호: 마지막 + 1)
2. 모든 필드 채우기 (문제 정의, 범위, 비범위 필수)
3. 복잡한 작업: 인간 검토 요청 후 진행
4. 완료 후: 이 파일의 상태 및 완료일 업데이트
