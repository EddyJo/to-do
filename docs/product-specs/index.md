## 핵심 제품 원칙 (PRD 요약)

1. **회피 우선 정렬**: `avoidance_score` 높은 Todo를 먼저 표시
2. **메모 → 액션 변환**: AI가 메모/회의록에서 Todo 추출
3. **사용자 통제권**: AI 추출 결과는 항상 draft — 승인 없이 반영 금지
4. **통합 흐름**: 메모와 Todo를 하나의 흐름으로 연결
